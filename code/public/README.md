# Static assets

Place the following image files here. They are referenced by the site but were
NOT included in the original export (only remote mochausercontent.com URLs were
available, which cannot be downloaded offline). Supply the real images:

- `logo.png`      — site logo (header, footer, email, favicon, og:image). ~ transparent PNG.
- `hero.png`      — homepage hero background image.
- `logo-alt.jpg`  — alternate logo (reserved).

Until these files are added, image areas will render as broken/empty.
